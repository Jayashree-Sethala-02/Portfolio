Silver Layer SQL — What Actually Runs
// identical to what I'd write in real Microsoft Fabric
-- Step 1: Bronze → Silver -- Clean messy SSIS data INSERT INTO silver_holdings SELECT UPPER(TRIM(account_id)) AS account_id, asset_class, ROUND(market_value, 2) AS market_value, date, 'silver' AS layer FROM bronze_holdings WHERE account_id IS NOT NULL AND market_value > 0 AND date IS NOT NULL GROUP BY UPPER(TRIM(account_id)), asset_class, date; -- Step 2: Silver → Gold -- Aggregate for analytics INSERT INTO gold_portfolio SELECT s.account_id, a.platform, s.asset_class, SUM(s.market_value) AS market_value, t.total_portfolio, ROUND(SUM(s.market_value) * 100.0 / t.total_portfolio, 2) AS allocation_pct FROM silver_holdings s JOIN account_info a ON s.account_id = a.account_id JOIN ( SELECT account_id, SUM(market_value) AS total_portfolio FROM silver_holdings GROUP BY account_id ) t ON s.account_id = t.account_id GROUP BY s.account_id, s.asset_class;

Drift Detection SQL
// runs against Gold layer
-- Probity · Daily Drift Detection -- Runs 7:00 AM via Azure Functions WITH drift_calc AS ( SELECT g.account_id, a.client_name, g.platform, g.asset_class, ROUND(g.allocation_pct, 2) AS current_pct, ROUND(t.target_pct, 2) AS target_pct, ROUND(g.allocation_pct - t.target_pct, 2) AS drift_pct, ABS(g.allocation_pct - t.target_pct) AS abs_drift FROM gold_portfolio g JOIN target_allocation t ON g.account_id = t.account_id AND g.asset_class = t.asset_class JOIN account_info a ON g.account_id = a.account_id ) SELECT *, CASE WHEN abs_drift > 10 THEN 'URGENT' WHEN abs_drift > 5 THEN 'REBALANCE' ELSE 'ON TARGET' END AS status FROM drift_calc ORDER BY abs_drift DESC;
Analytics · Performance Reporting
Performance Report — YTD 2024
Time-Weighted Returns vs S&P 500 benchmark. TWR removes the effect of client cash flows — industry standard for RIA reporting. Data sourced from Gold layer.
📘
TWR (Time-Weighted Return) is used here because it isolates the portfolio manager's decisions from when clients deposit or withdraw money. This is the industry standard for RIA performance reporting. Alpha = YTD Return − Benchmark Return.
